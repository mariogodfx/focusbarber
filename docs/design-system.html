Animations

Motion behaviors implemented across the landing page.

Scroll Reveal

Trigger
IntersectionObserver with threshold: 0.1, rootMargin: '0px 0px -50px 0px'
CSS class
.reveal — initial state: opacity: 0; transform: translateY(30px); transition: all 0.7s ease
Visible state
.reveal.visible — opacity: 1; transform: translateY(0)
Delay variants
.reveal-delay-1 (0.1s), .reveal-delay-2 (0.2s), .reveal-delay-3 (0.3s) — applied as transition-delay

Marquee

Effect
Continuous horizontal scroll of brand names / text strip
Keyframe
@keyframes marquee { 0% { transform: translateX(0); } 100% { transform: translateX(-50%); } }
Duration
~30s linear infinite
Font
Bebas Neue, uppercase, large size (clamp 2-4rem)
Implementation
Duplicate content inline (marquee-content + clone) for seamless loop

Hero Text Animation

Effect
Heading lines clip-reveal from bottom with translateY + clip-path
Keyframe
@keyframes revealText { 0% { clip-path: inset(0 0 100% 0); transform: translateY(60px); } 100% { clip-path: inset(0); transform: translateY(0); } }
Stagger
Each line has incremental animation-delay (0.5s, 0.8s, 1.1s)
Duration
~0.8s ease-out each, applied on page load

Hero Image Parallax

Effect
Hero image shifts vertically on scroll at 15% rate
Implementation
window.addEventListener('scroll') → transform: scale(1.05) translateY(scrolled * 0.15 px)
Constraint
Only active while scrollY < window.innerHeight

Counter Animation

Effect
Numbers count up from 0 to target value when stats section enters viewport
Trigger
IntersectionObserver with threshold: 0.5
Implementation
setInterval at 25ms, increment = target / 60 (approx 1.5s duration)
Unobserve
Observer unobserves element after animation completes (one-time)

Navigation Scroll Effect

Trigger
scrollY > 80px adds .scrolled class to #nav
Effect
Adds background + blur, subtle border-bottom

Testimonial Auto-rotate

Interval
5 seconds via setInterval
Transition
opacity 0.5s ease
Interaction
Click dot → show that testimonial, reset auto-rotate timer

Smooth Scroll

Effect
All anchor links (href="#section") scroll smoothly via JS
Implementation
element.scrollIntoView({ behavior: 'smooth', block: 'start' })
