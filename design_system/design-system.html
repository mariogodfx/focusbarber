Forms

COMPONENTES DE Formulário

O único formulário real é o .booking-form. Inputs/select date/time; sem textarea, checkbox, radio, toggle, file ou DatePicker custom. Estados existentes: default, placeholder, focus (borda gold). Sem estado de error/disabled no CSS próprio.

default + placeholder

focus (borda gold)

Label: display:block; font-size:0.65rem; letter-spacing:0.25em; text-transform:uppercase; color:var(--gray-light); margin-bottom:0.5rem. Nenhum helper-text / error-text existe no source.

4.2 — Select (dropdown nativo)

-webkit-appearance:none, mesma estilização do input (border-bottom), cursor:pointer. Opções usam o popup nativo (select option { background:var(--black); color:var(--white) }). Sem dropdown custom.

date / time nativos

Inputs nativos type=date / type=time, mesma regra do input. Sem componente de calendário custom.
